# Codex Rules For This Repo

- Default to Chinese when reporting to the user, while keeping commands, file names, API names, and project names in English.
- Follow Repo Map First: `.ai/` first, then `codegraph status --json .`, then candidate files, then minimal reads.
- Since this repo is mostly Markdown, healthy CodeGraph may report `fileCount=0`; do not rebuild just because there are no indexed symbols.
- Preserve the distinction between factual attribution and practical engineering value.
- Do not copy the full guideline text into global/project rules unless explicitly requested; prefer compact, non-duplicative summaries.
- Keep Claude, Cursor, and skill/plugin variants in sync if changing the actual four principles.
